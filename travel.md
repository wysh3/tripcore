# Travel Agency

## Features

1. Dashboard → Show Overview  
   1. Total Users  
   2. Total Bookings  
   3. Total Revenue  
   4. Packages  
   5. Destinations  
   6. Pending Bookings  
   7. Recent activity  
   8. Quick actions  

2. Homepage  
   1. Hero Section Settings  
      a. Background image  
   2. Feature section  
      a. Banner image  

3. Booking  
   1. Total Bookings  
   2. Total Revenue  
   3. Confirmed  
   4. Pending  
   5. Completed  
   6. All Bookings (0)  
   7. Enquiries  
      a. Total Enquiries  
      b. New  
      c. Contacted  
      d. Converted  

4. Packages  
   1. Package title  
   2. Destination (Drop down)  
   3. Duration  
   4. Tour types ⇒ Selection option  
   5. Tour category ⇒ Drop down  
   6. Max persons  
   7. Accommodation  
   8. Cities/Route  
   9. Age Range  
   10. On off button ⇒ Trip Customizable, Best price, flights included  
   11. Main package image  
   12. Map/Route Image  
   13. About Tour  
      a. Short description  
      b. Highlights  
      c. Special Notes  
      d. Tags  
   n. Pricing  
      a. selling price  
      b. original price  
      c. discount  
   o. Day-wise itinerary  
      a. Image(optional) ⇒ pdf of package  
      b. Day1 ⇒ admin can more days  
         a. Title  
         b. Duration  
         c. Description  
      c. Inclusion & exclusion  
         a. Inclusion ⇒ can add one line  
         b. Exclusion ⇒ can add one line  
         c. Cancellation Policy ⇒ can add one line  
         d. Gallery Images ⇒ can upload multiple images  
   p. Departure Information  
      a. Departure Type  
      b. Departure Points  
      c. Upcoming Departure Dates ⇒ can add multiple  
         a. start date,  
         b. seats left  
         c. price  

5. Destination  
   1. Country name  
   2. City/State (optional)  
   3. Region ⇒ Asia, europe etc (optional)  
   4. Description  
   5. Destination image  

6. Users  
   admin can change the role and delete the users records  
   1. total users  
   2. admin users  
   3. regular users  

7. Sales  
   1. Sale name  
   2. url slug  
   3. description  
   4. hero image  
   5. status ⇒ active or not  

8. Review Management  
   1. Total Reviews  
   2. Average rating  
   3. 5 star reviews  
   4. recent Reviews  
   Above are shown as stats  
   1. Add Review ⇒ admin can add edit or delete it  
      a. Customer Name  
      b. Package  
      c. Destination  
      d. Rating ⇒ drop down ⇒ 1 - 5 stars  
      e. Video Testimonial  

9. Car Rental  
   CRUD operation  
   1. Full Name  
   2. WhatsApp Number  
   3. Email  
   4. Pickup and drop  
   5. Start Date and End Date  
   6. Adults and Children  
   7. Type of Car  
      Sedan (2-3 persons)  
      SUV / Innova  
      Tempo Traveller  
      Luxury Car  
   8. Driver Language  
      a. English  
      b. Spanish  
      c. Hindi  
   9. Trip Type  
      a. One Way  
      b. Round Trip  
      c. Multi-City Tour
